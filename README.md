# CS6350
### Spring 2026
### Professor: Anurag Nagar
### Course Description:
The course will focus on data mining and machine learning algorithms
for analyzing very large amounts of data or Big data. Map Reduce and
No SQL system will be used as tools/standards for creating parallel
algorithms that can process very large amounts of data.
The course material will be drawn from textbooks as well as recent
research literature. The following topics will be covered this year:
Hadoop, Mapreduce, NoSQL systems (Cassandra, Hive, BigTable,
HBASE, SPARK), Large scale supervised machine learning, Data
streams, Clustering, and Applications including recommendation
systems, Web and security.
